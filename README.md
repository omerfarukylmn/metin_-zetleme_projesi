# Metin Özetleme Projesi

Bu proje, doğal dil işleme dersi kapsamında geliştirilen bir metin özetleme uygulamasıdır.

## Ekip Üyeleri
- Atakan Ünal
- Ömer Faruk Yelman
- Mert Demirkol

## Proje Amacı
Kullanıcının girdiği metni daha kısa ve anlamlı bir özet haline getiren bir sistem geliştirmek.

## İlk Hedef
- Proje klasör yapısını oluşturmak
- Gerekli kütüphaneleri kurmak
- Basit bir arayüz hazırlamak
- İlk özetleme altyapısını kurmak
